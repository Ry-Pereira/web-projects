# 🎲 Dice Roller Project

## 📌 Overview

The Dice Roller Project is a web application built with HTML, CSS, and JavaScript that simulates rolling one or more six-sided dice.

Users can enter the number of dice they want to roll, click the Roll Dice button, hear a dice rolling sound effect, view the numerical results, and see images representing each die rolled.

---

## ✨ Features

🎲 Roll between 1 and 16 dice

🔢 Display the value of every die rolled

🖼️ Show an image for each rolled die

🔊 Play a dice rolling sound effect

🎨 Custom styled interface using CSS

📱 Responsive webpage structure

## 🚀 How to Run

1. Download or clone the project files.
2. Ensure the images and audio folders are included.
3. Open index.html in a web browser.
4. Enter the number of dice to roll.
5. Click the Roll Dice button.
6. View the results and dice images.
