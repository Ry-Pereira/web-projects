# ⏱️🌲 StopWatch Website 🌲⏱️

A customizable stopwatch website featuring lap tracking, start/pause/restart controls, and bright 🌞 / dark 🌙 mode themes.

---

## 📖 Description

The StopWatch Website allows users to:

✨ Start the stopwatch  
⏸️ Pause the stopwatch  
🔄 Restart the stopwatch  
🏁 Record lap times  
📋 View recorded laps in a scrollable container  
🌞 Switch to bright mode  
🌙 Switch to dark mode  
🎨 Enjoy a custom-designed user interface with themed forest backgrounds

---

## ✨ Features

### ⏱️ Stopwatch Controls
- ▶️ Start timer
- ⏸️ Pause timer
- 🔄 Restart timer
- 🕒 Live time updates

### 🏁 Lap Tracking
- ➕ Record lap times
- 📋 View all laps
- 🔢 Automatic lap numbering
- 📜 Scroll through recorded laps

### 🎨 Appearance
- 🌞 Bright Mode
- 🌙 Dark Mode
- 🌲 Forest-themed backgrounds
- 🔤 Custom ElmsSans font
- ✨ Smooth color transitions
- 🖱️ Interactive button hover effects
