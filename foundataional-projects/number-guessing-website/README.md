# 🎲🔢 Number Guessing Game Website

## 📌 Overview

This project is a simple HTML, CSS, and JavaScript number guessing game. The program generates a random number between 1 and 100, and the user attempts to guess the correct number.

The game provides feedback after each guess, letting the user know whether their guess is too high, too low, or correct. It also tracks the number of attempts made before the correct number is guessed.

Users can:

* Guess a randomly generated number between 1 and 100
* Receive feedback after each guess
* Track the number of attempts made
* Practice logical thinking and number estimation
* Learn JavaScript event handling and DOM manipulation

## ▶️ How to Run the Project

1. Download or clone the repository:

   ```bash
   git clone <your-repository-url>
   ```

2. Open the project folder.

3. Verify the following files exist:

   * `index.html`
   * `styles.css`
   * `script.js`

4. Open:

   ```text
   index.html
   ```

5. Enter a number between **1 and 100** and click **Submit Guess**.

6. Continue guessing until you find the correct number.
