# 🔢➖🔄➕ Number Counter Website

## 📌 Overview

This project is a simple HTML, CSS, and JavaScript webpage that allows users to interact with a number counter. Users can increase, decrease, or reset the displayed number using buttons on the webpage.

The project demonstrates basic DOM manipulation, event handling, and webpage styling for beginner web developers.

Users can:

* Increase the counter value
* Decrease the counter value
* Reset the counter to zero
* Learn basic JavaScript event handling
* Explore a simple interactive webpage

## ▶️ How to Run the Project

1. Download or clone the repository:

   ```bash
   git clone <your-repository-url>
   ```

2. Open the project folder.

3. Ensure the following files are present:

   * `index.html`
   * `styles.css`
   * `script.js`

4. Run the webpage by opening:

   ```text
   index.html
   ```

5. Interact with the buttons to increase, decrease, or reset the counter.
