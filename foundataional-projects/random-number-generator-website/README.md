# 🎲🔢 Random Number Generator Website

## 📌 Overview

This project is a simple HTML, CSS, and JavaScript webpage that generates a random number between 1 and 100 when the user clicks a button.

The project demonstrates basic JavaScript concepts such as event handling, DOM manipulation, and random number generation.

Users can:

* Generate a random number between 1 and 100
* Display the generated number on the webpage
* Learn JavaScript event handling
* Practice DOM manipulation
* Explore basic web development concepts

## ▶️ How to Run the Project

1. Download or clone the repository:

   ```bash
   git clone <your-repository-url>
   ```

2. Open the project folder.

3. Verify the following files exist:

   * `index.html`
   * `styles.css`
   * `script.js`

4. Open:

   ```text
   index.html
   ```

5. Click the **Generate Random Number** button to generate a number between 1 and 100.
