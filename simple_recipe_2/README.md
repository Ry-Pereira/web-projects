# 🍪 Simple Recipe 2

This is a simple HTML and CSS project that presents a sugar cookie recipe. It includes ingredients, step-by-step instructions, styling, and an image to create a visually organized recipe webpage.

---

## 🧱 Project Structure

```
Simple_Recipe_2/
│
├── index.html
├── styles.css
└── images/
    └── sugar_cookie_photo.jpg
```

---

## 🚀 Features

- Clean recipe webpage layout  
- Ingredients list  
- Step-by-step instructions  
- Styled using external CSS  
- Custom font from Google Fonts  
- Centered image with border styling  
- Organized sections using div containers  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Google Fonts (EB Garamond)  
- External image file  

---

## ▶️ How to Run the Project

1. Download or clone the repository:

```bash
git clone <your-repository-url>
```

2. Open the project folder.

3. Make sure the following files exist:

```
index.html
styles.css
images/sugar_cookie_photo.jpg
```

4. Open:

```
index.html
```

5. View the recipe webpage in your browser.

## 📂 Project Contents
