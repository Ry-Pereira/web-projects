# 🐊🐻🐱🐶🐘🦊🦒🐴🦎🐆🦘🦁 Animal Gallery

## 📌 Overview

This project is a simple HTML and CSS Animal Gallery webpage that displays a collection of animals using styled image containers. Each animal has a static image that changes to an animated version when hovered, creating an interactive and engaging experience.

Users can:
- View different animals in a gallery layout
- Hover over animals to see animations
- Explore a responsive, styled grid-like design
- Learn basic HTML and CSS structure

This project is designed to:

1. Display a variety of animals in a structured gallery

## ▶️ How to Run the Project

1. Download or clone the repository:
   ```bash
   git clone <your-repository-url>
   ```

2. Open the project folder.

3. Ensure the `images` folder contains:
   - `backgrounds/nature_background.jpg`
   - `still/` images
   - `moving/` GIF files

4. Run the webpage by opening:
   ```text
   index.html
   ```

5. Hover over animals to see animations.
